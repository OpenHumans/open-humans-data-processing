Open Humans Data Export for 23andme data

This tool will access your 23andme account and, with your permission, receive a copy of your full 23andme genotyping data set.

To do this, we will need to request access to the following:

  • Basic information
    Profile identifiers and whether each profile has been genotyped. (Some accounts have profiles for more than one individual.)
  • Names
    Names help you select which profile in the account is yours (i.e. the one you are exporting). Even for accounts with only one profile, your name is an important validation to confirm that you are exporting the correct data set. These names are not recorded in your Open Humans data set, they are only accessed to facilitate your export of the 23andme data.
  • Full genotype
    The purpose of this export tool is to access this data and process it to create a data set that may be managed by you within Open Humans. We will only export genotyping data for the profile you select.
